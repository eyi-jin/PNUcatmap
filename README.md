---------------동작 방식---------------------
* MainActivity

- 구글 로그인

* HomeActivity2
- 이메일 로그인 시 이동하는 액티비티

* SignupActivity
 -MainActivity에서 회원가입 버튼 누르면 이동하는 액티비티
 
 ---------------추가 오류발견------------------
 - 여러 개의 액티비티가 밑에 깔려있어서 종료시 계속 닫아야함

 - 로그인시 비밀번호 설정창과, 회원가입시 비밀번호 설정창 다름(-> 숫자만 나오는것, 영어입력도 가능한 것)
